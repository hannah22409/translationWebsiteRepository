Servicios




Ofrezco una variedad de servicios de traducción, incluyendo los siguientes:

  • Traducción de correspondencia personal, como cartas, correos electrónicos o publicaciones en redes sociales
  • Traducción de materiales de marketing, como copias de sitios web, descripciones de productos o esloganes publicitarios
  • Traducción de subtítulos para videos o películas cortas
  • Traducción de menús u otros materiales para restaurantes o negocios que atienden a una clientela bilingüe
  • Traducción de publicaciones de blog o artículos sobre una variedad de temas
  • Localización de aplicaciones o interfaces de software para que sean más fáciles de usar para una audiencia de habla inglesa
  • Transcripción y traducción de audio o video, para podcasts, entrevistas u otros medios

¡y mucho más!
